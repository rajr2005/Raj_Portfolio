## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
cd raj-portfolio
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Start the development server**
```bash
npm start
# or
yarn start
```

The application will open in your browser at `http://localhost:3000`

### Build for Production

```bash
npm run build
# or
yarn build
```

This creates an optimized production build in the `build/` folder.

## 🎨 Customization

### Change Colors
Edit the gradient colors in your CSS files. Primary gradient colors:
- Primary: `#667eea` to `#764ba2`
- Modify in any `.css` file to update the theme

### Update Content
Edit the component files directly:
- `Hero.js` - Change hero section content
- `About.js` - Update about information
- `Skills.js` - Modify skills and proficiency levels
- `Projects.js` - Add or remove projects
- `Contact.js` - Update contact information

### Add Social Links
Update the social links in:
- `Hero.js` - Hero section social icons
- `Contact.js` - Contact section social links
- `Footer.js` - Footer social links
